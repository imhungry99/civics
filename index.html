como funciona

todas semanas uma dica será lançada na página dica semanal, após a dica ser lançada, o seu objetivo é desvendar o enigma, se você conseguir, você irá encontrar um código, com este código você irá para a página onde a dica foi lançada, para enviar sua resposta.
O último time que enviar ou o time que não enviar será eliminado
